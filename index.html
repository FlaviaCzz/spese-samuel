Come aggiornare i dati su tutti i dispositivi

1. Usa l'app normalmente e inserisci le spese a mano.
2. Quando vuoi salvare tutto, tocca Scarica CSV qui sopra.
3. Rinomina il file scaricato da spese_samuel.csv a dati.csv.
4. Vai su github.com/FlaviaCzz/spese-samuel → Add file → Upload files → carica dati.csv → Commit changes.
5. Riapri l'app da qualsiasi dispositivo — i dati sono aggiornati ovunque.
